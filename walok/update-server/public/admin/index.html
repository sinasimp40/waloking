Build From Uploaded Source

Upload the launcher and/or server source as a .zip — this server installs dependencies, rebrands per customer (logo + brand name + subtitle), runs electron-builder, packs the resulting payload, and ships it to the chosen channel(s). Any online installs are notified instantly. Leave a slot empty to skip that side: launcher-only, server-only, or both. Progress streams to the build consoles below.

Version must be strictly newer than the chosen target's currently shipped version for the role being uploaded (launcher-only uploads are not blocked by a newer server version, and vice-versa). To re-issue, bump the version.
